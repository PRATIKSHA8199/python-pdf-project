--Customer Service Analysis--

**About the Project**
This project analyzes customer complaints data to understand how efficiently services are handled.
The dataset contains more than 3.6 lakh (364,000+) records of complaints.

**Objective**
Find the most common complaints
Check how much time is taken to solve complaints
Compare service performance
Understand complaint patterns across cities

**Tools Used**
Python
Pandas
NumPy
Matplotlib
Seaborn

**Steps Performed**
Loaded the dataset
Cleaned missing values
Converted date columns into proper format
Removed incorrect data
Created new column: Request Closing Time

**Analysis**
Top complaint types
Complaints per city
Response time for each complaint
Visual graphs (bar chart, scatter plot)

**Statistical Tests**
T-test
Kruskal-Wallis test
p-values

**Conclusion**
This project shows how data analysis helps improve customer service by identifying delays and patterns.
